# Course Devops Workshop :: Docker and Kubernetes

## Course outline
* Start with Why and Problems
* DevOps
  * People -> Process -> Tools
  * Team Topologies 
    * Good and ant-patterns
  * Tools

  * Kubernetes
  * Helm chart
